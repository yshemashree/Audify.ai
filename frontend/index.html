Capabilities

What Audify does

01
🧠

Prompt Expansion

Mistral-7B turns your one-word input into a vivid, detailed sonic description with texture, space, and intensity.

02
🔍

Semantic Search

Vector embeddings match your prompt against a library of real recordings. Close match? Instant result — no wait.

03
⚡

AI Generation

AudioLDM2 diffusion model synthesises a unique audio clip from scratch in ~30 seconds when no match is found.

04
🎨

No Limits

Sounds that don't exist. Emotions. Fictional worlds. Describe anything and Audify will try to render it into audio.

Process

How it works

01

Elaborate Agent

Your input hits Mistral-7B Instruct which expands it — "cat" becomes "a soft indoor meow from a domestic shorthair, close-mic, slight reverb."

02

Search Agent

The description is embedded and compared to real audio clips via cosine similarity. Score ≥ 0.75 = instant match returned.

03

Generation Agent

No match? AudioLDM2 takes the elaborated description and generates a brand new wav file using audio diffusion.

04

Orchestration

smolagents coordinates all three, handles retries, parses JSON output, and delivers description + audio to the UI.
