Process

How Audify works

Three specialized AI agents collaborate in real-time to produce your sound.

01

Prompt Elaboration Agent

Your short input — "cat" or "storm" — is passed to Mistral-7B Instruct, which expands it into a precise, vivid description: textures, environment, intensity, and spatial feel.

02

Semantic Search Agent

The elaborated description is embedded via Sentence-Transformers and compared against a library of real sound clips using cosine similarity. If a match scores above 75%, it's returned immediately.

03

Audio Generation Agent

If no library match exists, the AudioLDM2 diffusion model takes the detailed description and synthesises a completely unique audio clip from scratch — in ~30 seconds.

04

Orchestration & Delivery

smolagents coordinates all three agents automatically, handles retries, parses results, and delivers the final audio with its source and description back to you in one clean response.

Built with AudioLDM2 · smolagents · Mistral-7B · FastAPI
